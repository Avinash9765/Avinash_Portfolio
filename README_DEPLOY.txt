AVINASH PORTFOLIO - GITHUB PAGES READY

Folder structure:
Avinash_Portfolio_GitHubPages/
├── index.html
└── assets/
    ├── avinash.png
    └── Avinash_Resume.docx

How to run locally:
1. Open this folder in VS Code.
2. Right-click index.html and choose Open with Live Server.
   OR double-click index.html.

How to deploy on GitHub Pages:
1. Create a new GitHub repository, for example: Avinash-Portfolio
2. Upload index.html and the assets folder.
3. Go to Repository Settings > Pages.
4. Select Source: Deploy from a branch.
5. Select Branch: main and Folder: /root.
6. Click Save.

Live URL format:
https://YOUR_USERNAME.github.io/Avinash-Portfolio/

Important:
- Do not rename assets/avinash.png unless you also update the image path in index.html.
- GitHub Pages is case-sensitive.
